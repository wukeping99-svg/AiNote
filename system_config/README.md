# ⚙️ system_config

这个文件夹放模板仓库的补充配置，避免把 `CLAUDE.md` 写得过长。

## 包含内容

| 文件 | 用途 |
| ---- | ---- |
| `lessons_learned.md` | 记录长期偏好和复用经验 |
| `templates.md` | 常见笔记模板 |
| `README.md` | 当前说明文件 |

## 使用原则

- `CLAUDE.md` 只保留高频、稳定、必须每次都读的规则
- 更细的模板和偏好放在这里
- 如果你继续扩展系统，可以在这里增加 `workflows.md`、`integrations.md` 等补充文件
