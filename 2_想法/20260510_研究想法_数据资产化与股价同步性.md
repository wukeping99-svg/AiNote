## 研究问题

数据资产化是否会降低股价同步性？

## 理论基础

### 理论机制

#### 机制1：信息透明度提升
- **理论基础**：Roll (1988) "噪音交易"理论 + Jin & Myers (2006) 信息透明度模型
- **逻辑链条**：数据资产化 → 信息透明度提升 → 更多私有信息进入股价 → 个股特质信息增加 → 股价同步性降低

#### 机制2：投资者异质性
- **理论基础**：Du (2003) 异质信念理论
- **逻辑链条**：数据资产化 → 信息渠道增加 → 投资者获取信息差异增大 → 交易行为分化 → 股价同步性降低

#### 机制3：市场有效性强
- **理论基础**：EMH的弱式有效
- **逻辑链条**：数据资产化 → 信息反应更快更充分 → 历史信息被充分挖掘 → 股票收益更独立于市场 → 同步性降低

## 变量设计

### X：数据资产化程度
| 指标 | 来源 | 说明 |
|-----|------|------|
| 数字化投入/营业收入 | 年报附注 | 信息技术投资 |
| 研发投入密度 | CSMAR | R&D支出/总资产 |
| 数字化转型指数 | 文本分析 | 陈德球(2021)等方法 |
| 无形资产占比 | CSMAR | 数据类无形资产 |

### Y：股价同步性
- **经典度量**：Roll (1988) & Jin & Myers (2006)
$$Sync_i,t = R^2_i,t$$
- **对数形式**：$\ln(R^2/(1-R^2))$
- **数据来源**：CSMAR或锐思数据库
- **计算方法**：
  - 市场模型回归：$R_{i,t} = \alpha + \beta R_{m,t} + \varepsilon_{i,t}$
  - $R^2$ 为拟合优度

### 控制变量
- 企业规模 (ln总资产)
- 资产负债率
- 盈利能力 (ROA)
- 股权集中度
- 换手率
- 机构投资者持股
- 行业、年份固定效应

- **行业**：制造业vs服务业
- **地区**：东部vs中西部

```stata
* 方法2：自己计算

gen R = ln(P/P[_n-1]) if stkcd == stkcd[_n-1] & trd_nd == trd_nd[_n-1]

* Step2: 计算市场收益率（流通市值加权）

* Step3: 分股票回归
bysort stkcd: reg R mkt_R, noc
gen R2 = e(r2)

* Step4: 生成同步性指标
gen Sync = ln(R2 / (1 - R2))
replace Sync = . if R2 == 1 | R2 == .
```

### 数字化转型指数构建

参考陈德球(2021)方法：
1. 抓取年报"管理层讨论与分析"文本
2. 构建数字化关键词词表：人工智能、大数据、云计算、物联网、区块链等
3. 计算词频作为数字化转型指数

```python
# Python文本分析
import jieba
import re

keywords = ['数字化', '智能化', '大数据', '云计算', '人工智能', '物联网', 
            '区块链', '数字经济', '工业互联网', '5G']

def digital_index(text):
    words = jieba.lcut(text)
    count = sum([1 for w in words if w in keywords])
    return count / len(words) * 1000
```

## 稳健性检验

### 1. 替换核心变量
- 使用研发投入密度代替数字化投入
- 使用滞后一期数字化指数
- 使用 $R^2$ 原始值代替对数形式

### 2. 内生性处理
- **工具变量**：地区互联网普及率、地区数字经济发展水平
- **滞后变量**：L.digital（克服同期偏差）
- **DID**：2019年数字经济政策试点

### 3. 样本调整
- 剔除金融行业
- 剔除ST股票
- 剔除IPO前三年
- 缩尾处理 (1% winsorize)

| H1 | β1 < 0 | 数据资产化降低同步性 |

| H2b | 中介效应显著 | 投资者行为渠道 |
| H3a | |β民企| > |β国企| | 民企效应更强 |

## 主要结论

1. **假设验证**：β1显著为负？
2. **经济意义**：数字化提高1个标准差，同步性下降多少？
3. **机制渠道**：哪些中介变量显著？
4. **异质性**：哪些企业效果更明显？
